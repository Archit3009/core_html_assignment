HTML TAGS

  1. html-> The root element of an HTML page, encapsulating the entire content.
  2. head-> Contains meta-information about the document, such as title, links to stylesheets, and scripts.
  3. title->Specifies the title of the document displayed in the browser's title bar or tab.
  4. body->Encloses the main content of the document, including text, images, videos, and other elements visible to the user.
  5. div->Defines a division or section within an HTML document, often used for layout purposes and grouping content.
  6. a->Creates hyperlinks to other web pages, files, locations within the same page, or email addresses.
  7. img-> Inserts images into the document, with attributes specifying the image source, dimensions, alt text, etc.
